# Why It's Great
Brainboard bridges the gap between diagramming and coding. It allows you to design your architecture visually and automatically generates the Terraform code for it.

# Key Free Features
- **Design**: Visual architecture designer.
- **Code Generation**: Auto-generate Terraform code from diagrams.
- **Projects**: Free for personal projects.
- **Reverse Engineering**: Import existing TF files to visualize them (limited in free tier).
