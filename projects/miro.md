# Why It's Great
Miro is the industry standard for digital whiteboarding. It's essential for remote brainstorming, architecture diagrams, and agile planning.

# Key Free Features
- **Boards**: 3 editable boards.
- **Collaborators**: Unlimited team members.
- **Templates**: Access to a vast library of templates (Kanban, User Mapping, Flowcharts).
- **Integrations**: Core integrations with Slack, Trello, etc.
- **Canvas**: Infinite canvas.
