# Why It's Great
Discord has evolved from a gaming chat app to a powerful communication tool for developer communities and teams. It supports markdown, code highlighting, and screen sharing.

# Key Free Features
- **Users**: Unlimited members and messages.
- **Voice/Video**: Reliable low-latency voice and video calls (unlimited).
- **Screen Sharing**: Use "Go Live" to share your screen with low latency.
- **Integrations**: Extensive bot ecosystem and webhooks for Dev/Ops alerts.
