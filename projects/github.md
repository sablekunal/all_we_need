# Why It's Great
GitHub is the standard for code hosting. Its free tier is incredibly generous, offering unlimited public/private repositories, built-in CI/CD (Actions), and project management tools.

# Key Free Features
- **Repositories**: Unlimited public and private repositories.
- **Collaborators**: Unlimited collaborators on public & private repositories.
- **CI/CD**: 2,000 free automation minutes/month (GitHub Actions).
- **Storage**: 500MB package storage.
- **Community**: Access to the largest open-source community in the world.
