# Why It's Great
Bitbucket is optimized for professional teams, especially those using the Atlassian suite (Jira, Confluence).

# Key Free Features
- **Users**: Unlimited public repositories; up to 5 users on private repositories.
- **Pipelines**: 50 build minutes/month.
- **Storage**: 1 GB file storage (LFS).
- **Integration**: Best-in-class integration with Jira Software.
