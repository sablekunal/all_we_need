# Why It's Great
Postman simplifies every step of the API lifecycle. It allows you to design, mock, debug, test, document, monitor, and publish APIs.

# Key Free Features
- **Collaboration**: Teams of up to 3 users.
- **Requests**: Unlimited API requests and collections.
- **Mock Servers**: 1,000 mock server calls/month.
- **Monitoring**: 1,000 monitoring calls/month.
- **Public API Network**: Publish documentation to the world's largest public API network.
